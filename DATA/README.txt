In this folder we have 3 folders: 

- TestSetAngles: Here, we have for each RNA in our test set, a file with the ID, the sequence and all the alpha angles for each nucleotide. (30 files)
- TrainingSetAngles: Here, we have for each RNA in our training set, a file with the ID, the sequence and all the alpha angles for each nucleotide. (192 files)
- sample: In this folder, we have a file, with an example of a fasta sequence and a folder where we have the one hot encoded matrix (npy format) of this sequence.
